RabbitMQ
========

Start a [rabbitmq docker image](https://hub.docker.com/_/rabbitmq/)
	```sh
	docker run -d --hostname rabbit --name rabbit -p 15672:15672 -p 5672:5672 rabbitmq
	```sh
